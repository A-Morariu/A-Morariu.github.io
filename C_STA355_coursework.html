\[ F^{-1} (\tau) = \lambda^{-1}\log(1-\tau) \]

Recall the CDF of the exponential distribution: \[ F(x) = 1 - e^{-\lambda x} \] And find the inverse for some fixed x. \[ F^{-1}(x) = -\frac{1}{\lambda} \log(1-x) \] As such for some constant quantile we have: \[ F^{-1}(\tau) = -\frac{1}{\lambda} \log(1-\tau) \]

qq.expo <- function(x,plot=T, tle) {
             x <- sort(x)  # sorts the data from smallest to largest
             n <- length(x) # length of x
             tau <- (c(1:n)-3/8)/(n+1/4)
             if (plot) {
                   plot(-log(1-tau),x,xlab="Standard exponential quantiles",
                         ylab="Ordered data", main = tle)
                   abline(a=0,b=mean(abs(x)))
                  # draws a line with slope equal to
                  # the sample mean and intercept 0
                   }
             else {
               r <- list(x=x,quantiles=-log(1-tau))
               r
               }
}

: \[ F^{-1} (\tau) = \sigma (-\log(1-\tau))^\frac{1}{\alpha} \]

Following the same steps as Question 1, the proof is trivial.

By lecture note 5, slide 5:

\[ Var(X_{(k)}) = \frac{\tau (1-\tau)}{n f^2( \sigma\left[-log(1-\tau)\right]^\frac{1}{\alpha})} \] Where f is the Weibull function. After some cancellation we get the function:

\[ Var(X_{(k)}) = \frac{\tau \sigma^2}{n \alpha^2(1-\tau) \left[ \log(1-\tau)\right]^{2(1-\frac{1}{\alpha})}} \]

From lecture note 6 slide 6, we know that the exponential distribution is parameterized by: \[ \lambda = f(F^{-1}(\tau)) \] And furthermore, we can leverage part A of this question to find the desired rate parameter. As such we plug in part A into the Weibull distribution: \[F^{-1}(\tau) = \sigma (-\log(1-\tau))^\frac{1}{\alpha} \longrightarrow f(x; \alpha, \sigma)\] \[ \Rightarrow f(F^{-1}(\tau)) = \left( \frac{\alpha}{\sigma} \right) \left( \frac{\sigma (-\log (1 - \tau))^\frac{1}{\alpha}}{\sigma} \right)^{\alpha - 1} exp\left[ - \left( \frac{\sigma (-\log (1 - \tau))^\frac{1}{\alpha}}{\sigma} \right)^\alpha \right] \]
and after some cancellation we get: \[ \Rightarrow \lambda = \frac{\alpha (1-\tau)}{\sigma} \left[ \log(1-\tau) \right] ^{1 - \frac{1}{\alpha}}\]

large_alphas <- c(1:10)
small_alphas <- c(1:10)/10
tau_values <- c(0:99)/100
par(mfrow=c(2,5))
for (alpha in large_alphas) {
    means <- (alpha*(1-tau_values)*(-log(1-tau_values))^(1-1/alpha))
plot(means, main = paste("alpha = ", as.character(alpha)))
}

for (alpha in small_alphas) {
    means <- (alpha*(1-tau_values)*(-log(1-tau_values))^(1-1/alpha))
plot(means, main = paste("alpha = ", as.character(alpha)))
}

We notice that the graphs of the means exhibit one of 2 behaviours depending on the value of alpha. For alpha = 1 we get a constant slope on the means. For alpha < 1, we can see that the means exhibit an exponentially decreasing trend. While for alpha > 1, a unimodal distribution with a around the 50th quantile that shifts towards the right as alpha increases. The reason that this is important is that this distribution can be used to model real life events. It acts similarly to the hazard function where, based on the shape of the graph, we can tell if a the event is more likely to occur closer to time 0 or further in time (ex. time of death, time until failure of a lightbulb). It references the “better new than old” vs. “better old than new” principle discused in class.

\[ \frac{d}{dx} \left(t - \frac{1}{\mu (F)} \int_0^x F^{-1}(s)fs \right) =1 - \frac{1}{\mu (F)} F^{-1} (x)\] \[ \Rightarrow 0 =1 - \frac{1}{\mu (F)} F^{-1} (x) \Rightarrow \boxed{ \mu (F) = F^{-1} (x)} \]

\[ \mathcal{P}(F) = t - \frac{1}{\mu} \int_0^x F^{-1}(s)ds \] \[ = \int_0 ^x 1ds - \frac{1}{\mu} \int_0^x F^{-1}(s)ds\] \[ = \frac{1}{\mu} \int_0 ^x \left( \mu - F^{-1}(s) \right) ds\] We leverage the value for x when the function is maximized. \[ = \frac{1}{\mu} \int_{F(-\infty)} ^{F(\mu)} \left( \mu - F^{-1}(s) \right) ds \] Now we perform a change of variables such that: \[ s = F(x) \Rightarrow ds = F'(x)dx = f(x)dx \] And we get the following (note change on the bounds of the integral). \[ = \frac{1}{\mu} \int_{-\infty} ^{\mu} \left[ \mu f(x) - F^{-1}(F(x))f(x) \right] ds \] \[ = \frac{1}{\mu} \int_{-\infty} ^{\mu} \left( \mu - x \right)f(x)dx \] Now we just need to manipulate this to get the expectation as given in the question. \[ = \frac{1}{2\mu} \left( \int_{-\infty} ^\mu (\mu-x)f(x)dx + \int_\mu^\infty(x-\mu)f(x)dx \right) \] \[ = \frac{1}{2\mu} \int_{-\infty}^{\infty} \vert \mu - x \vert f(x)dx = \frac{\mathbb{E} \left[ \vert \mu - x \vert \right]}{2\mu}\]

For a substitution principle based estimator we use the following: \[ \mu(\hat{F}) = \bar{X} \]

And the formula from Part B becomes: \[ \mathcal{P}(F) = \frac{\mathbb{E}(\vert X-\bar{X}\vert)}{2\bar{X}} = \frac{\sum \vert X-\bar{X}\vert}{2\bar{X}} \]

salaries <- scan("Datasets/STA355/incomes.txt", sep = " ")

pietra_index <- function(X) {
  mu <- mean(X)
  return(sum(abs(X - mu)) / (2*length(X)*mu))
}

jackknife <- function(theta_hat, X) {
        n <- length(X)
        theta_minus_i <- rep(0, n)
        for (i in 1:n) {
          theta_minus_i[i] <- theta_hat(X[-i])
        }
        theta_dot <- mean(theta_minus_i)
        jackvar <- (n-1)/n * sum((theta_minus_i - theta_dot)^2)
        return(sqrt(jackvar))
}
